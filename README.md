# The Ink Lady Website

Premium dark-luxury one-page website for The Ink Lady tattoo and piercing studio in Prishtinë, Kosovo.

## Files

- `index.html` English SEO page
- `sq.html` Albanian SEO page
- `de.html` German SEO page
- `privacy-policy.html` standard privacy policy page
- `assets/tailwind.css` local compiled Tailwind CSS
- `script.js` language switching, gallery filters, lightbox, mobile menu, booking form
- `robots.txt` and `sitemap.xml`

## Before publishing

Replace `https://your-domain.com/` in the HTML files, `privacy-policy.html`, `robots.txt`, and `sitemap.xml` with the final domain.

Replace placeholder gallery SVGs with real tattoo and piercing photos when ready. Keep descriptive alt text for SEO.

The booking form opens WhatsApp using `+383 48 46 46 45`.

The Privacy Policy is a standard template and should be reviewed before publishing if strict legal compliance is required.
